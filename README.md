# Pedro Rivera - Portfolio

Repositorio del sitio personal / portfolio estático.

Qué contiene
- Archivos estáticos en `src/` (HTML, CSS, JS, assets).
- Configuración de build (`webpack.config.js`, `webpack-prod.config.js`).

Instrucciones rápidas
1. Ignorar dependencias locales: ya existe `.gitignore` con `node_modules/` y `dist/`.
2. Para subir a GitHub:
   - Crea un repo en GitHub y añade el remoto:
     `git remote add origin https://github.com/<usuario>/<repo>.git`
   - Empuja la rama principal:
     `git push -u origin main`

Si quieres, puedo crear el repositorio en tu cuenta usando la CLI `gh` si está instalada y autenticada.
